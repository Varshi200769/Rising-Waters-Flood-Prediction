# Project Planning Phase

## Project Objectives

- Develop a flood prediction system using Machine Learning.
- Analyze historical flood data.
- Compare multiple classification algorithms.
- Deploy the best-performing model using Flask.
- Provide a simple prediction interface.

## Project Workflow

1. Dataset Collection
2. Data Analysis
3. Data Preprocessing
4. Feature Scaling
5. Model Training
6. Model Evaluation
7. Model Saving
8. Flask Development
9. Testing
10. Documentation

## User Stories

The project was developed using multiple user stories covering data collection, preprocessing, model building, evaluation, deployment, testing, and documentation.
